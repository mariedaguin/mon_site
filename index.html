Data mining

The multiplication of information requires the development and the use of techniques allowing to extract useful knowledge from it. To make good decisions, it's essential to apprehend and use correctly all the information. Analysis of multidimensional data and in particular data mining, which seeks to transform data into more structured knowledge, provide adapted tools to this type of problem.

I've studied different data sets by putting in practice a lot of analysis methods, using R :
- Introduction: examples of problem of prediction, supervised and unsupervised training. 
- Multidimensional statistics: random vectors and descriptive statistics 
- Principal component analysis 
- Factorial discriminant analysis 
- Clustering : k-means algorithm and agglomerative hierarchical clustering algorithm of Ward 
- Mixture model and clustering
- Methods of regression.

DataWarehouse Ex-Nihilo

A data warehouse is a system used for reporting and data analysis. DWs are central repositories of integrated data from one or more disparate sources.  
I've learnt how to build a data warehouse ex-nihilo, in batch mode processing, with Oracle. I've studied the ETL (Extract, Transform, Load) process and used technologies as UML, SQL and PL/SQL.

I've made some projects including the Fantastic one that I'm going to describe below.

The company Fantastic sells mainly science fiction, thriller, books and movies. To this end, she has several stores in big cities in France. The direction wishes to make a wide study on the sales of the last year in order to introduce new strategic orientations (opening new stores, etc). I've realized (using Oracle) a data warehouse to make the company able de take decisions.
